# 🤝 Contributing to Handwritten Digit Recognizer

Thank you for taking the time to contribute! This project is open to improvements of all kinds — bug fixes, new features, documentation updates, or better preprocessing approaches.

---

## 📋 How to Contribute

### 1. Fork the Repository

Click the **Fork** button at the top right of this repository page.

### 2. Clone Your Fork

```bash
git clone https://github.com/YOUR_USERNAME/Handwrittenrecognizer.git
cd Handwrittenrecognizer
```

### 3. Create a Feature Branch

```bash
git checkout -b feature/your-feature-name
```

Use a descriptive branch name, e.g.:
- `feature/add-cnn-model`
- `fix/image-inversion-bug`
- `docs/improve-readme`

### 4. Make Your Changes

- Keep code clean and readable
- Add comments where logic is non-obvious
- Update the README if your changes affect usage

### 5. Commit Your Changes

```bash
git add .
git commit -m "feat: add CNN model for improved accuracy"
```

Follow [Conventional Commits](https://www.conventionalcommits.org/) where possible:
| Prefix | Use For |
|---|---|
| `feat:` | New features |
| `fix:` | Bug fixes |
| `docs:` | Documentation updates |
| `refactor:` | Code restructuring without behavior change |
| `test:` | Adding or updating tests |

### 6. Push and Open a Pull Request

```bash
git push origin feature/your-feature-name
```

Then open a **Pull Request** from your fork to `main`. Describe what you changed and why.

---

## 🐛 Reporting Issues

If you find a bug or have a feature request, please [open an issue](https://github.com/pedadarohan-dot/Handwrittenrecognizer/issues) with:

- A clear title and description
- Steps to reproduce (for bugs)
- Expected vs actual behavior
- Your Python and library versions

---

## 💡 Ideas for Contribution

- Implement a CNN-based model and compare accuracy
- Add a web interface (Gradio / Streamlit)
- Support for drawing digits in the browser
- Model export and serialization with `joblib`
- Unit tests for preprocessing functions
- Experiment with different optimizers or architectures

---

## 📜 Code of Conduct

Be respectful, constructive, and kind. All contributions are valued regardless of experience level.
